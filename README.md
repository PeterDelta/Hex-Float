**Hex<->Float** Convertidor de hexadecimal a punto flotante y viceversa utilizando el estándar **IEEE-754 de precisión simple (32 bits)**.
Facilita el análisis, depuración y modificación de datos utilizados por aplicaciones, motores gráficos, videojuegos, etc.

## Funciones principales

* Conversión de **Hexadecimal → Float (IEEE-754 32 bits)**.
* Conversión de **Float → Hexadecimal (IEEE-754 32 bits)**.
* Cálculo inmediato del resultado hexadecimal tras aplicar operaciones de **multiplicación** o **división** mediante un factor configurable.
* Conversión entre números **hexadecimales** y **decimales enteros de 64 bits**.

## Características técnicas

* Conversión binaria directa mediante reinterpretación de bits IEEE-754, garantizando una correspondencia exacta entre el valor hexadecimal y su representación en memoria.
* Soporte completo para todos los patrones de 32 bits, incluidos:

  * Valores normales.
  * Valores subnormales.
  * ±0.
  * ±Infinity.
  * NaN.

**Hex<->Float** Hexadecimal to floating point converter and vice versa using the standard **IEEE-754 single-precision (32-bit)**.
It facilitates the analysis, debugging, and modification of data used by applications, graphics engines, video games, etc.

## Main Functions

* Conversion from **Hexadecimal → Float (IEEE-754 32-bit)**.
* Conversion from **Float → Hexadecimal (IEEE-754 32-bit)**.
* Immediate calculation of the hexadecimal result after applying **multiplication** or **division** operations using a configurable factor.
* Conversion between **hexadecimal** and **64-bit integer decimals**. * Automatic updating of all results as you type.

## Technical Features

* Direct binary conversion using IEEE-754 bit reinterpretation, guaranteeing an exact match between the hexadecimal value and its memory representation.
* Full support for all 32-bit patterns, including:

  * Normal values.
  * Subnormal values.
  * ±0.
  * ±Infinity.
  * NaN.
